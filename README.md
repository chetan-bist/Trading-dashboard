# trading-dashboard

A clean, dark-themed trading analytics dashboard for visualizing sales, revenue, profit, and sessions. No database required — just clone and run the frontend.

---

## Features

* Dark UI with KPI cards and charts
* Responsive layout (desktop-focused)
* Demo data (no backend / no DB)

---

## Quick Start

```bash
# Clone the repo
git clone https://github.com/<your-username>/trading-dashboard.git
cd trading-dashboard

# Install dependencies
npm install
# or
pnpm install

# Run dev server
npm run dev
# or
pnpm dev
```

Open `https://trading-dashboard-vert.vercel.app/` in your browser.

---

## Tech Stack

* React / Next.js (frontend)
* Tailwind CSS
* ApexCharts or Chart.js (charts)

---

## Notes

* This repo is frontend-only for demo/visual purposes. No database or backend included.
* Replace demo/static data with your API when ready.
